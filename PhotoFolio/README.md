# Visuals by IamMojo - PhotoFolio

Professional photography and cinematography portfolio website for Akintolure MojolaOluwa, based in Lagos, Nigeria. This site showcases visual storytelling for events, brands, and portraits.

## Features
- Responsive gallery of photography works
- About, Services, and Contact pages
- Social media links (WhatsApp, Instagram)
- Modern UI with Bootstrap, Swiper, Glightbox, and AOS animations
- Contact form (with PHP mailer integration)

## Technologies Used
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Swiper.js (gallery slider)
- Glightbox (image lightbox)
- AOS (animations)
- PHPMailer (for contact form email)

## Getting Started
1. **Clone the repository:**
   ```sh
   git clone https://github.com/clas-6/visualsbymojo.git
   ```
2. **Open `index.html` in your browser** to view the site locally.
3. **To deploy:**
   - You can host on Vercel, Netlify, or any static hosting platform.
   - For contact form email, configure `email_config.php` and PHPMailer as needed (see `backup_php_files/` for setup).

## Folder Structure
- `assets/` - Images, CSS, JS, vendor libraries
- `about.html`, `services.html`, `contact.html`, `index.html` - Main site pages
- `backup_php_files/` - PHP email setup (optional)

## Deployment
- Push to GitHub and connect to Vercel for automatic deployment.
- No build step required; static HTML/CSS/JS.
